# Um programinha e um tutorial de como utilizar o OpenCV com Python
OpenCV é a principal biblioteca de código aberto para a visão computacional, processamento de imagem e aprendizagem de máquina, apresenta também a aceleração de GPU
para operação em tempo real.
- Ao instalar o Python, caso tenhamos marcado a opção Add Python PATH, o Pip também estará instalado (precisaremos dele).

# Com isso em mente, basta abrir o terminal Windows (cmd), e digitarmos o comando:pip install opencv-python.
# No linux o comando será: sudo pip3 install opencv-python.

No código estará a sequência de comandos, além de comentários usados para o aprendizado, que você poderá usar para testar o programa.
